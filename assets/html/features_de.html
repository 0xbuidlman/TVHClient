Information
TVHClient beinhaltet viele Funktionen, die jetzt und in Zukunft weiterhin frei verwendet werden können. Jedoch ist die Entwicklung und das Hinzufügen von neuen Funktionen (teilweise von Usern gewünscht) zeitaufwändig. Deshalb habe ich entschieden eine Bezahlfunktion einzubauen. Für einen kleinen Betrag werden zusätzliche Funktionen aktiviert. Durch den Kauf zeigst Du Deine Unterstützung und förderst die weitere Entwicklung. Vielen Dank!
Wenn Du den Kaufen Button auswählst erscheint ein neues Fenster. Dort kannst Du Deinen Kauf bestätigen und evtl. nochmal Dein Google Passwort eingeben. Nachdem der Bezahlvorgang abgeschlossen ist, kann das Fenster geschlossen werden. Es sind anschließend alle Funktionen freigeschaltet.
Zusätzliche Funktionen
  • Streaming Profile
    Diese sind die neueren Wiedergabe- und Aufnahmeprofile, die vom Server bereit gestellt werden. (Der Server muss diese Funktion unterstützen. Prüfe im Status Bildschirm ob die API-Version des Servers mindestens 16 zeigt.)
  • Mehr Kontrolle über Serienaufnahmen
    Neue Serienaufnahmen können nun mit mehr Parametern erstellt und bestehende editiert werden
  • Unterstützung für manuelle Aufnahmen
    Definiere selber welche Programme wann und wie aufgenommen werden sollen. (Der Server muss diese Funktion unterstützen. Prüfe im Status Bildschirm ob die API-Version des Servers mindestens 17 zeigt.)
