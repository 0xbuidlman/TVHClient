Information
TVHClient beinhaltet viele kostenlose Funktionen. Dies wird auch in Zukunft so bleiben. Für einen kleinen Betrag können aber zusätzliche Funktionen freigeschaltet werden. Durch den Kauf zeigst Du Deine Unterstützung und förderst die weitere Entwicklung. Mit dem Kaufen Button erscheint ein neuer Dialog in dem Du den Kauf bestätigen mußt. Nach Abschluss des Bezahlvorganges kann das Fenster geschlossen werden. Es sind anschließend alle Funktionen freigeschaltet.
Vielen Dank!
Zusätzliche Funktionen
  • Streaming Profile
    Diese sind die neueren Wiedergabe- und Aufnahmeprofile, die vom Server bereit gestellt werden.
    (Der Server muss diese Funktion unterstützen. Im Status Bildschirm muss die API-Version des Servers mindestens 16 sein.)
  • Mehr Kontrolle über Serienaufnahmen
    Erstelle exaktere Serienaufnahmen mit Hilfe zusätzlicher Parameter wie z.B. Start- und Endzeit, Dauer, Tag der Aufnahme und mehr. Ausserdem können bestehende Serienaufnahmen editiert werden
    (Der Server muss diese Funktion unterstützen. Im Status Bildschirm muss die API-Version des Servers mindestens 13 sein.)
  • Unterstützung für manuelle Aufnahmen
    Bestimme selbst, welche Programme wann und wie aufgenommen werden. Definiere diese Aufnahmen durch Parameter wie z.B. Start- und Endzeit, Tag der Aufnahme und mehr. (Der Server muss diese Funktion unterstützen. Im Status Bildschirm muss die API-Version des Servers mindestens 17 sein.)
  • Suchen während der Eingabe
    Im Suchbildschirm startet die Suche bereits wenn 3 Zeichen eingegeben werden. Die Suchergebnisse aktualisieren sich selbst wenn sich die Sucheingabe ändert.
  • Aufnahmen bearbeiten
    Bearbeiten nachträglich Programme die geplant sind oder bereits aufgenommen werden. Zusätzliche Parameter wie z.B. Start- und Stopzeit, Vorlauf- oder Nachlaufzeit, Priorität und weitere können verändert werden.
    (Der Server muss diese Funktion unterstützen. Im Status Bildschirm muss die API-Version des Servers mindestens 21 sein.)
