Information
TVHClient contains a variety of features that are free to use and will remain free. The ongoing development and adding new features (partially requested by users) is time consuming. So I have decided to add a in-app purchase option. For a small fee you can enable additional features. It will also ensure the further development and shows your support. Thank you!
If you select the purchase menu a new dialog will open where you will be prompted to confirm the purchase. You might be prompted for you Google password. Once the purchase is done the dialog can be closed and all features will be available.
Additional Features
  • Streaming profiles
    These are the newer playback and recording profiles that are provided by the server. (Your server must support this feature. Check the status page that your servers API version is at least 16.)
  • More control over series recordings
    Add new series recordings with a lot more parameters. Also edit already defined series recordings.
  • Support for timer recordings
    Manually define programs to record. (Your server must support this feature. Check the status page that your servers API version is at least 17.)
